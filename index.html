Smart Campus Mobile

Platform Asisten Akademik Berbasis Progressive Web App

UAS Web Mobile Project

Aplikasi web mobile-first yang dirancang khusus untuk mengintegrasikan kebutuhan data akademik mahasiswa secara instan, interaktif, dan responsif.

  • SPA Navigation: Transisi menu cepat tanpa reload.
  • Smart Schedule: Render data jadwal kuliah dinamis dari JSON.
  • Mission Tracker: Penyimpanan data tugas via IndexedDB.
  • PWA Ready: Dapat di-install di HP & support Offline Mode.
  • Hardware API: Pendeteksi koordinat Geolocation & Dark Mode.

Tech Stack

Tailwind CSS v4 jQuery & AJAX IndexedDB Service Worker
